How to start PHP's built-in web server:

Double-click the DOS Prompt here and type:

> php -S localhost:8000
   
   (Press Ctrl-C to stop the web server)

Double-click the "Local host at 8000" to open the index.php 

Then launch the php lesson codes in web browser, e.g.
http://localhost:8000/samplecodes/lesson02/inthebody.php
